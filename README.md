# Strata Notion

A Strata component for working with the Notion API from typed Go tasks.

This repository packages a reusable component binary plus shared Go definitions so a Strata app can:

- authenticate against Notion with an integration token
- search for pages
- query a data source
- create pages
- append blocks to an existing block or page
- open `https://notion.com` from the host

## Project Layout

- [`main.go`](main.go) implements the component handlers
- [`api.go`](api.go) contains the HTTP client used for Notion requests
- [`definition/definition.go`](definition/definition.go) exposes the typed component definitions your app imports
- [`definition/blocks.go`](definition/blocks.go), [`definition/objects.go`](definition/objects.go), and [`definition/properties.go`](definition/properties.go) provide helper types for composing Notion payloads

The component manifest is named `notion` and is currently version `1.0.0`.

## What It Exports

The shared `definition` package exposes these operations:

- `Setup`: stores the API key in the component process and verifies it with `GET /v1/users/me`
- `FindPage`: searches Notion with `POST /v1/search`
- `QueryDataSource`: runs `POST /v1/data_sources/{id}/query`
- `CreatePage`: creates a page with optional parent, properties, markdown, and children
- `Append`: appends child blocks with `PATCH /v1/blocks/{id}/children`
- `LaunchNotion`: asks the host to open `https://notion.com`

It also includes a few typed helpers for building requests:

- `PropertyTitle`, `PropertyRichText`, `PropertyEmail`, `PropertyCheckbox`, `PropertyDate`, `PropertyDateRange`
- `PlainText`, `CodeText`
- `BlockParagraph`, `BlockCallout`

## Requirements

- Go `1.26`
- A Strata app/runtime
- A Notion integration token

`SetupInput.APIKey` is sent directly as the `Authorization` header, so pass the full value you want Notion to receive.

## Using It In A Strata App

Add the dependency to your app with:

```bash
go get github.com/jacksonzamorano/strata@v1.0.0
go get github.com/jacksonzamorano/strata-notion@latest
```

Import the shared definitions package in your app and import the component into the runtime:

```go
package main

import (
	"os"
	"path"

	"github.com/jacksonzamorano/strata"
)

func main() {
	cd, _ := os.Getwd()

	rt := strata.NewRuntime([]strata.Task{
		// your tasks here
	}, strata.Import(
		strata.ImportLocal(path.Join(path.Dir(cd), "strata-notion")),
	))

	panic(rt.Start())
}
```

For local development against sibling checkouts, you can still use `replace` directives:

```go
replace github.com/jacksonzamorano/strata-notion => ../strata-notion
replace github.com/jacksonzamorano/strata => ../strata
```

Then call the component from a task:

```go
package main

import (
	"os"

	notion "github.com/jacksonzamorano/strata-notion/definition"
	"github.com/jacksonzamorano/strata"
)

func createStatusPage(input strata.RouteTaskNoInput, ctx *strata.TaskContext) *strata.RouteResult {
	_, ok := notion.Setup.Execute(ctx.Container, notion.SetupInput{
		APIKey: os.Getenv("NOTION_API_KEY"),
		Debug:  false,
	})
	if !ok {
		return strata.RouteResultError("failed to authenticate with notion")
	}

	page, ok := notion.CreatePage.Execute(ctx.Container, notion.CreatePageInput{
		ParentPageId: "your-parent-page-id",
		Properties: map[string]any{
			"Title": notion.PropertyTitle(notion.PlainText("Daily status")),
		},
		Children: []any{
			notion.BlockParagraph(notion.PlainText("Generated by Strata")),
			notion.BlockCallout("blue_background", "i", notion.CodeText("synced")),
		},
	})
	if !ok {
		return strata.RouteResultError("failed to create page")
	}

	return strata.RouteResultSuccess(page)
}
```

## Notes

- `CreatePageInput` accepts one parent target at a time: `ParentDataSourceId`, `ParentDatabaseId`, or `ParentPageId`
- `FindPage` returns `PagesOutput`, which includes page ids, timestamps, URLs, and property data
- When `SetupInput.Debug` is true, the component logs raw request and response payloads through the component logger
- The client currently sends Notion API version `2025-09-03`

## Running This Repo

Build the component with:

```bash
go build .
```

In normal use, Strata launches the component for you after you import it into an app runtime.
